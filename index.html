개인정보처리방침

최종 업데이트: 2025년 10월 30일

1. 개요

Voice Notion App("본 앱")은 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 본 앱이 사용자의 정보를 어떻게 수집, 사용, 보호하는지 설명합니다.

2. 수집하는 정보

본 앱은 다음과 같은 정보를 수집합니다:

  • API 키 정보: 사용자가 입력한 Notion API 키 및 Gemini API 키는 기기 내부에만 저장됩니다.
  • 음성 데이터: 음성 인식을 위해 일시적으로 처리되며, 기기 외부로 전송되지 않습니다.
  • 텍스트 데이터: 변환된 텍스트는 사용자가 선택한 Notion 데이터베이스에만 저장됩니다.

본 앱은 사용자의 개인 식별 정보를 수집하지 않습니다.

3. 정보의 사용

수집된 정보는 다음 목적으로만 사용됩니다:

  • 음성을 텍스트로 변환
  • Gemini API를 통한 텍스트 요약
  • Notion API를 통한 데이터 저장

사용자가 입력한 API 키는 암호화되어 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다.

4. 정보의 공유 및 제3자 제공

본 앱은 사용자의 정보를 제3자와 공유하지 않습니다. 단, 다음의 경우는 예외입니다:

  • Notion API: 사용자가 입력한 텍스트를 Notion 데이터베이스에 저장하기 위해 Notion API를 사용합니다.
  • Gemini API: 텍스트 요약을 위해 Google Gemini API를 사용합니다.

이러한 API 사용은 사용자가 제공한 API 키를 통해 이루어지며, 본 앱 개발자는 사용자의 데이터에 접근할 수 없습니다.

5. 데이터 보안

본 앱은 사용자 데이터를 보호하기 위해 다음과 같은 조치를 취합니다:

  • API 키는 Android Keystore를 사용하여 암호화되어 저장됩니다.
  • 모든 API 통신은 HTTPS를 통해 암호화됩니다.
  • 음성 데이터는 기기 내에서만 처리되며 저장되지 않습니다.

6. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 데이터 삭제: 앱 설정에서 언제든지 저장된 API 키를 삭제할 수 있습니다.
  • 앱 제거: 앱을 제거하면 기기에 저장된 모든 데이터가 삭제됩니다.
  • 데이터 접근: 사용자의 모든 데이터는 사용자의 Notion 계정에 저장되며, 사용자만 접근할 수 있습니다.

7. 어린이 개인정보 보호

본 앱은 만 13세 미만의 어린이를 대상으로 하지 않으며, 의도적으로 어린이의 개인정보를 수집하지 않습니다.

8. 개인정보처리방침 변경

본 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있을 경우, 앱 내 공지를 통해 사용자에게 알립니다.

Privacy Policy

Last Updated: October 30, 2025

1. Overview

Voice Notion App ("the App") takes user privacy very seriously. This Privacy Policy explains how the App collects, uses, and protects user information.

2. Information We Collect

The App collects the following information:

  • API Keys: Notion API key and Gemini API key entered by users are stored only on the device.
  • Voice Data: Temporarily processed for speech recognition and not transmitted outside the device.
  • Text Data: Converted text is stored only in the user's selected Notion database.

The App does not collect personally identifiable information.

3. How We Use Information

Collected information is used only for the following purposes:

  • Converting voice to text
  • Summarizing text via Gemini API
  • Saving data via Notion API

API keys entered by users are encrypted and stored only on the device, never transmitted to external servers.

4. Information Sharing

The App does not share user information with third parties, except:

  • Notion API: Used to save user-entered text to Notion database.
  • Gemini API: Used for text summarization.

These API uses are conducted through user-provided API keys, and the App developer cannot access user data.

5. Data Security

The App takes the following measures to protect user data:

  • API keys are encrypted using Android Keystore.
  • All API communications are encrypted via HTTPS.
  • Voice data is processed only on the device and not stored.

6. User Rights

Users have the following rights:

  • Data Deletion: Users can delete stored API keys anytime in app settings.
  • App Removal: Uninstalling the app deletes all data stored on the device.
  • Data Access: All user data is stored in the user's Notion account, accessible only by the user.

7. Children's Privacy

The App is not intended for children under 13 and does not knowingly collect personal information from children.

8. Changes to Privacy Policy

This Privacy Policy may be updated as needed. Users will be notified of significant changes through in-app notifications.
